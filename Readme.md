# ⚡ AI-Powered Energy Consumption Forecasting

## 📌 Overview

This project predicts future electricity consumption using Machine Learning.
It helps smart cities, industries, and households optimize energy usage and reduce wastage.

---

## ❗ Problem Statement

Energy demand is unpredictable, leading to:

* Power wastage
* High electricity bills
* Poor energy planning

This project uses AI to forecast energy consumption based on past usage patterns.

---

Used in:

* Renewable Energy Systems

## 🧪 How It Works

1. Load historical energy data
2. Perform preprocessing
3. Extract features (hour, day, lag values)
4. Train ML model
5. Predict future energy usage
6. Visualize results
